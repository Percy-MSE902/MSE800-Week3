System for SQLit3
* The system manages students, instructors, and courses with the following operations:
    1. Add new students with ID, name, email, and date of birth.
    2. Add new instructors with ID, name, email, and expertise field.
    3. Create new courses with course code, name, description, and criteria.
    4. Enroll students in courses (Applied table) with applied date.
    5. Assign instructors to teach courses (Teaches table).
    6. Count the number of students in a specific course.
    7. List all instructors teaching a specific course.
